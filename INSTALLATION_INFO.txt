## üõ†Ô∏è Prerequisites

Before installing BudgeIT, ensure you have the following installed on your system:

### 1. Python Installation
- Download Python from [python.org](https://python.org)
- Ensure Python is added to your system PATH
- Verify installation by running: `python --version`

### 2. Package Manager (pip)

- Verify by running: `pip --version`

- Required if installing from source

### Method 1: Pre-built Executable (Recommended for most users)
- Download the latest release from our GitHub repository

## ‚ùì Troubleshooting Common Issues

### Installation Fails:
- Ensure Python is properly installed and in PATH
- Check internet connection for dependency downloads
- Run terminal/command prompt as administrator

### Permission Errors:
- Install as administrator/root if required
- Check write permissions for installation directory

### Missing Dependencies:
- Update pip: `python -m pip install --upgrade pip`
- Install dependencies manually: `pip install -r requirements.txt`
