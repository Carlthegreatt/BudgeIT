# 🎉 Welcome to BudgeIT!

## Installation Complete - Let's Get Started!

Congratulations! You've successfully installed **BudgeIT** - Your Personal Budget Management Application.

This guide will help you set up your account and start managing your finances effectively.

---

## 🚀 Quick Start Guide

### Step 1: Launch BudgeIT
- **Windows:** Double-click the BudgeIT desktop icon or find it in Start Menu
- **macOS:** Open Applications folder and click BudgeIT
- **Linux:** Run `python run.py` from the application directory

### Step 2: Create Your Account
1. Click **"Get Started"** on the welcome screen
2. Fill in your personal information
3. Create a secure password
4. Set up security questions (optional but recommended)

### Step 3: Initial Setup Wizard
The setup wizard will guide you through:
- **Currency Selection** - Choose your local currency
- **Income Setup** - Add your monthly income sources
- **Expense Categories** - Customize spending categories
- **Budget Goals** - Set your monthly budget limits

---

## 🎯 First-Time Configuration

### 📊 Dashboard Overview
Your main dashboard includes:
- **Budget Summary** - Current month's spending overview
- **Recent Transactions** - Latest income and expenses
- **Savings Tracker** - Progress toward your savings goals
- **Analytics** - Visual charts of your spending patterns

### 💰 Setting Up Your Budget
1. Navigate to **Budget** section
2. Click **"Create New Budget"**
3. Set monthly limits for each category:
   - Food & Dining
   - Transportation
   - Entertainment
   - Utilities
   - Healthcare
   - Shopping
   - Others (customizable)

### 💳 Adding Your First Transaction
1. Click the **"+"** button or **"Add Transaction"**
2. Select transaction type (Income/Expense)
3. Choose category
4. Enter amount and description
5. Set date (defaults to today)
6. Save transaction

## 🛡️ Security & Data Management

### Your Data is Safe:
- All financial data stored locally on your device
- No cloud storage or external server access
- Password-protected user accounts
- Optional data encryption

### Backup Recommendations:
1. **Regular Backups** - Weekly backup of your database
2. **Export Options** - Export data to CSV/Excel formats
3. **Location:** Database stored in `budgeit/accounts.db`
4. **Manual Backup** - Copy the entire BudgeIT folder

### Data Privacy:
- Zero data transmission to external servers
- No tracking or analytics collection
- Full control over your financial information
- Optional email notifications (local SMTP only)
